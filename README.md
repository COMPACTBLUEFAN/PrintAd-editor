# PrintAd Editor v2

**PrintAd Editor** — это удобное десктопное приложение для визуального создания и редактирования листовок, рекламных баннеров и коммерческих предложений. Оно позволяет выгружать готовые макеты в форматы **PDF**, **PNG** или **HTML**.

Приложение поставляется в виде портабельного файла `.exe`, не требующего установки, и имеет интерфейс профессионального, но простого визуального конструктора.

## 📁 Структура проекта

```text
PrintAd/
├── PrintAd Editor v1.exe # Готовое приложение (просто запустите его)
├── README.md             # Инструкция, которую вы сейчас читаете
├── Templates/            # Папка для ваших базовых HTML-макетов
│   ├── cache/            # Папка для локальных ресурсов (картинки, фоны)
│   └── flyer_template.html
└── SourceCode/           # Исходный код приложения (для разработчиков)
```

## 🚀 Как пользоваться редактором

1. Запустите файл `PrintAd Editor v1.exe`.
2. В верхней панели нажмите **"1. Папка с шаблонами"** и укажите директорию с вашими базовыми макетами (по умолчанию это папка `Templates`).
3. В выпадающем списке выберите нужный макет. Он сразу загрузится на холст.

### 🎓 Интерактивное обучение

При первом запуске программы (или по нажатию кнопки **"❓ Помощь"** в правом верхнем углу) вам будет предложено пройти интерактивное обучение. Доступно два режима:
* **Базовый (1 мин):** быстрый обзор интерфейса и логики работы.
* **Продвинутый:** подробное объяснение всех функций (сохранение в HTML, прозрачный фон, панели GrapesJS).

### Редактирование на холсте (WYSIWYG)

В новой версии редактора мы избавились от боковых форм. Теперь вы работаете прямо с макетом:
* **Изменение текста:** Просто кликните **дважды** по любому тексту на макете и начните печатать.
* **Масштабирование:** Используйте кнопки `-`, `100%`, `+` в верхней панели для удобной работы с мелкими деталями.
* **Свободное перемещение:** Любой элемент можно схватить мышкой и свободно передвинуть в любую точку холста (как в Figma или Canva).
* **Добавление новых блоков:** Справа нажмите на иконку **квадратов (Блоки)**. Перетащите нужный элемент (Текст, Изображение) прямо на холст в нужное место.
* **Стили и цвета:** Выделите элемент (один клик) и справа нажмите на иконку **кисточки (Стили)**. Там можно изменить шрифт, размер, цвет, отступы, фон и тени.
* **Работа с изображениями:** Добавьте блок "Изображение" или кликните дважды по существующей картинке. Откроется менеджер ресурсов, который запустит обычный **Проводник Windows**. Выберите любую картинку с компьютера — она автоматически появится на холсте.

### Сохранение результата

1. В верхней панели нажмите **"2. Куда сохранить"** и выберите папку на вашем компьютере.
2. Введите желаемое имя файла в текстовое поле.
3. Опционально: поставьте галочку **"Прозрачный фон"**, если хотите сохранить макет без белой подложки (идеально для формата PNG).
4. Нажмите одну из кнопок экспорта: **В PDF**, **В PNG** или **В HTML**.
   *(При экспорте все добавленные вами локальные картинки будут встроены в итоговый файл. Временные служебные файлы будут сложены в специальную папку `Tempt` внутри папки с вашими шаблонами).*

## 🎨 Как создавать свои базовые шаблоны

Теперь вам **не нужно** прописывать специальные `data-` атрибуты! 

Редактор построен на базе GrapesJS. Вы можете взять **любой валидный HTML-файл** со стилями, положить его в папку `Templates`, и он автоматически станет доступен для полного визуального редактирования.

## 🛠️ Для разработчиков

Приложение написано с использованием современных технологий:
* **Electron** — движок для десктопного приложения, IPC и взаимодействия с файловой системой.
* **React + Vite** — для построения пользовательского интерфейса.
* **GrapesJS** — мощный open-source Web Builder Framework, предоставляющий функционал Drag-and-Drop и Inline редактирования.

### Установка и запуск исходного кода
Убедитесь, что у вас установлен Node.js.
Перейдите в папку `SourceCode`:

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки (Vite HMR + Electron)
npm run dev

# Сборка портабельного .exe (сохранится в dist-electron/)
npm run build
```
