# 🎨 PrintAd Editor
**Ваша карманная студия для рекламных материалов**

Привет! 👋 Добро пожаловать в **PrintAd Editor** — супер-простую, но мощную программу, которая позволяет легко собирать и редактировать рекламные листовки, баннеры и коммерческие предложения. Никаких сложных установок, регистраций или подписок. Запустили — и вы уже работаете!

Вы можете легко выгрузить свои шедевры в **PDF** (чтобы отдать в типографию), **PNG** (чтобы закинуть в соцсети) или **HTML** (чтобы отдать разработчикам).

---

## 🚀 Как начать

Поскольку мы предоставляем исходный код, у вас есть два варианта:

**Вариант А: Быстрый старт (если у вас уже есть файл)**
1. Просто откройте готовый файл `PrintAd Editor.exe` (если вы скачали скомпилированную версию).
2. Выберите папку с вашими заготовками (например, `Templates`).
3. Редактируйте и сохраняйте!

**Вариант Б: Сборка из исходников**
Смотрите раздел **"Для разработчиков"** ниже. Вы можете сами скомпилировать `.exe` буквально парой команд!

## 🖌️ Как редактировать? Почти как в Canva!

Мы постарались сделать всё так, чтобы вам не пришлось долго разбираться:

* **Меняйте текст на лету:** Просто кликните по любому тексту **дважды** и пишите всё, что захотите.
* **Двигайте что угодно:** Хватайте мышкой любую картинку или заголовок и тащите в любое место на холсте. Свобода полная!
* **Заменяйте картинки:** Дважды кликнули по фотке — открылось обычное окно Windows. Выбрали свою картинку, и она автоматически вставилась в макет!
* **Стройте из блоков:** Справа есть панелька с плюсиком `➕`. Оттуда можно вытаскивать новые блоки прямо на холст: текст, секции, колонки.
* **Раскрашивайте:** Выделили элемент, нажали на иконку «кисточки» 🎨 справа — и меняйте цвета, шрифты, отступы и тени как душе угодно.
* **Нужен прозрачный фон?** Легко! Перед экспортом в PNG просто поставьте галочку «Без фона». Идеально для наклеек и логотипов!
* **Слишком мелко?** Используйте кнопки масштабирования сверху (`-` и `+`) или крутите колёсико мышки, зажав `Ctrl`. А кнопка **«Сброс»** вернет масштаб к идеальным 100%.

## 💡 Полезная фишка: Интерактивный помощник

Заблудились или хотите узнать больше фишек? В правом верхнем углу всегда висит кнопка **«❓ Помощь»**. 

Нажмите на неё, и программа интерактивно проведёт вас за ручку по всем кнопочкам и панелям. Можно пройти быстрый тур за 1 минуту, а можно нырнуть в «Продвинутое обучение» и узнать, как стать настоящим профи.

## 🛠️ Для разработчиков (что под капотом)

Если вы заглянули сюда за кодом, то вам нужна папка `SourceCode/`. 
Приложение собрано на современном стеке: **Electron + React + Vite + GrapesJS**.

Как запустить это чудо локально:

```bash
# 1. Зайдите в папку с исходным кодом
cd SourceCode

# 2. Установите зависимости
npm install

# 3. Запустите в режиме разработки (HMR работает!)
npm run dev

# 4. Если хотите собрать свой портабельный .exe файл
npm run build
```

Приятного использования и красивых листовок! ✨
